# Create Virtual Environement

```
virtualenv <Virtual enviroment name>
```

# Activating Virtual Environment

```
 .\Scripts\activate
```

# Installing Selenium in Virtual Environment

```
pip install selenium
```

# My Environment

```commandline
Python 3.11.4
pselenium-4.10.0
```
